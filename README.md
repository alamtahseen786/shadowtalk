<h1># ShadowTalk 👀</h1>

ShadowTalk is a modern anonymous confession website where users can freely share their thoughts, secrets, and feelings without revealing their identity. The project is built using HTML, CSS, JavaScript, and Firebase.

The website features a dark aesthetic blue-themed interface, responsive design, scrollable confession feed, and dynamic post functionality. Users can post anonymous confessions instantly, and the posts remain visible even after refreshing the page.

This project was created as a beginner-friendly web development project to practice responsive layouts, DOM manipulation, Firebase integration, and dynamic content handling.
